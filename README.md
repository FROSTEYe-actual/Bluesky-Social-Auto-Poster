\#Auto-Poster for Bluesky

\#Tested on...
* Rocky Linux 9.6
* Nginx 1.29.3
* MariaDB 10.6.23
* PHP 8.3.27
* WordPress 6.8.3

\#PHP extensions must needed are...
* dom
* json
* intl
* mbstring
> It is highly probable that the majority of servers that have installed PHP 8.x will already have these extensions, particularly in the case of Wordpress servers.
> It is unnecessary to undertake the installation process once more.
> In the event of uncertainty, it is advisable to consult with the designated administrator of the hosting service.

\#Features
* Write a new post or modify existing post in Wordpress, plugin throw same one into Bluesky Social.
